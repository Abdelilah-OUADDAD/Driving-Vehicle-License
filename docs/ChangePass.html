Change Password

Pour modifier le mot de passe, cliquez sur Account Setting, puis sur Change Password.

Sur cet écran, nous voyons les informations concernant l'utilisateur et les champs de texte permettant de modifier le mot de passe.

Sur cet écran, lorsque nous remplissons le champ de texte pour changer le mot de passe et que nous cliquons sur le bouton Enregistrer, un message de confirmation s'affiche pour indiquer que l'opération a réussi
