Manage People Form

Ce formulaire présente la liste des personnes qui souhaitent obtenir leur permis de conduire.

Dans le même formulaire, nous pouvons filtrer par prénom, nom, identifiant, etc. Voici une image illustrant cette étape

Lorsque je clique sur le bouton droit, une liste contenant Afficher s'affiche : Show Details, Add New Person, Edit, Delete, Send Email, Phone Call.

Lorsque clique sur context Show Details Ce formulaire affiche les informations de la personne et comporte un lien permettant de modifier ces informations.

Lorsque clique sur context Update Person Ce formulaire affiche les informations dans une zone de texte pour les modifications et l'enregistrement.

Lorsque clique sur le bouton Ajouter ou sur le context , un formulaire intitulé Ajouter une nouvelle personne s'affiche. Dans ce formulaire, nous pouvons ajouter la personne pour laquelle nous souhaitons passer l'examen de conduite

Lorsque je clique sur « Supprimer » dans le contexte, une boîte de dialogue s'affiche pour confirmer ou non la suppression.
