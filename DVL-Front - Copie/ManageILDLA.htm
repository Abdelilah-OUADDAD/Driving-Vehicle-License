Manage International Driving License Applications

Manage International License

Pour accéder à la International license Application, cliquez sur Application, puis sur Manage Application, et enfin sur International license Application.

Lorsque l'on clique sur « International License Applications », un écran de gestion des licences internationales s'affiche.

Dans cette étape, nous voyons la fonctionnalité de filtrage.

Dans cette étape, nous voyons la liste du contexte que nous avons:Show Person Details, Show License Details, Show Person License History.

Dans l'option Show Person Détails, les informations concernant la personne apparaissent.

Dans l'option Show License Details, les informations de license international concernant la personne apparaissent.

Lorsque nous cliquons sur le option Show License History, l'écran « Historique des licences » apparaît et les informations sont affichées dans le tableau  International License History
