Manage Test Types

Lorsque je veux accéder à Manage Test Types, je clique sur Application.

Lorsque je clique sur Manage Test Types, cet écran apparaît.

Dans cette étape, nous voyons la liste du contexte que nous avons: Edit Test Types

Lorsque nous cliquons sur Modifier le type de demande, un écran apparaît nous permettant de modifier le titre, le description et les frais des types de Test.
